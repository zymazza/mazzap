Drag and drop files/folders, then choose a type per item: LiDAR, Footprints, Soils (SSURGO), Hydrology, or Photogrammetry.

Hydrology supports shapefile components such as .shp, .shx, .dbf, and .prj.

SSURGO uploads should include the full Web Soil Survey export folder structure (for example spatial/, tabular/, optional soildb_US_2003.mdb).

LiDAR is renamed to lidar_input.*. Footprints are renamed to Footprints.gdb. Photogrammetry keeps original names.

Uploading LiDAR and/or Footprints auto-runs processing for DEM, shrubs, trees, and building footprints; SSURGO and hydrology uploads are clipped to DEM extent when available.
